# Syphon PC Optimizer

**The ultimate Windows 10/11 performance toolkit.**

Syphon PC Optimizer applies 65+ system-level optimizations in one click — boosting FPS, reducing input lag, speeding up boot times, and cleaning up Windows.

## What it does

| Category | Description |
|----------|-------------|
| **CPU** | Unlocks full turbo boost, disables core parking, sets High Performance power plan |
| **GPU** | Enables HW-accelerated GPU scheduling, disables fullscreen stutter |
| **RAM** | Disables memory compression, optimizes page file, enables large system cache |
| **Disk** | Disables indexing and last-access timestamps, enables write caching |
| **Network** | Disables Nagle's algorithm, stops P2P bandwidth sharing, optimizes TCP |
| **Gaming** | Disables mouse acceleration, enables Game Mode, forces high precision timer |
| **Power** | Disables USB selective suspend and PCIe link state power saving |
| **Services** | Kills telemetry, Search indexer, SysMain, Cortana |
| **Privacy** | Disables error reporting, activity history, advertising ID, location tracking |
| **Visual** | Disables window animations, Snap Assist, sets Best Performance mode |
| **Cleanup** | Wipes temp files, browser cache, update cache, prefetch, font cache, DNS flush |

## Features

- **One-click optimize** — apply all 65+ tweaks instantly
- **Category control** — enable/disable individual optimization groups
- **System restore point** — automatically created before any changes
- **100% reversible** — built-in restore to undo everything
- **Disk benchmark** — test read/write speeds with integrated DiskSpd
- **Lightweight** — single portable exe, no installation required
